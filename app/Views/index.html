1. What is Mocker?

Mocker is a lightweight mock API generator built on top of Faker.js. It allows you to define response schemas dynamically using query parameters.

2. How to Use

Simply pass your schema as a query parameter ?s=. Each field follows the format: 

field:namespace.method(paramKey=value,...)

Separate one field with other using ;

You can also add:

  • ?c= ‚Üí to set the HTTP status code
  • ?h= ‚Üí to define custom headers (must be URL encoded)

Visit Faker.js documentation to see all function

Examples

Basic fields:

/api?s=id:string.uuid;name:person.fullName 
                
{
    "id": "c9b1b2de-1234-4e12-b5f6-98f23a8a1bcd",
    "name": "John Doe"
}

Nested fields (using dot .):

/api?s=user.id:string.uuid;user.name:person.fullName 

{
    "user": {
        "id": "9c5e2cda-5678-4eaa-b7a3-fb9a1e3a2bcd",
        "name": "Jane Smith"
    }
}

Arrays (using [N]):

/api?s=items[2].id:string.uuid;items[2].title:commerce.productName 

{
    "items": [
        {
            "id": "1d2c3b4a-aaaa-bbbb-cccc-111122223333",
            "title": "Incredible Granite Chair"
        },
        {
            "id": "4d5e6f7a-dddd-eeee-ffff-444455556666",
            "title": "Practical Wooden Table"
        }
    ]
}

With parameters:

/api?s=message:lorem.sentence(5);user.name:person.fullName(sex=male) 

{
    "message": "Lorem ipsum dolor sit amet consectetur.",
    "user": {
        "name": "Michael Johnson"
    }
}
