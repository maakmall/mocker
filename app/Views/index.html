What is Mocker

Mocker is a lightweight mock API generator built on top of Faker.js. It allows you to define response schemas dynamically using query parameters.

How to Use

Simply pass your schema as a query parameter ?s= with each field follows the format field:namespace.method(paramKey=value,...) , separate one field with other using ; and you can use any HTTP Method. You can also add:

  • ?c= â†’ to set the HTTP status code
  • ?h= â†’ to define custom headers (must be URL encoded)

Visit Faker.js documentation to see all function

Note : The helpers module from Faker.js is not supported

Use the schema builder to make it easier to create your schema.

Examples

Basic fields:

/api?s=id:string.uuid;name:person.fullName 

      {
        "id": "c9b1b2de-1234-4e12-b5f6-98f23a8a1bcd",
        "name": "John Doe"
      }

Nested fields (using dot .):

/api?s=user.id:string.uuid;user.name:person.fullName 

      {
        "user": {
          "id": "9c5e2cda-5678-4eaa-b7a3-fb9a1e3a2bcd",
          "name": "Jane Smith"
        }
      }

Arrays (using [n], default 1):

/api?s=category[3]:commerce.product;items[2].id:string.uuid;items[2].title:commerce.productName 

      {
        "category": [
          "Shoes",
          "Computer",
          "Soap"
        ],
        "items": [
          {
            "id": "626da13e-e0eb-4502-acae-7645138cbb59",
            "title": "Recycled Gold Soap"
          },
          {
            "id": "1fc075f2-9ed5-4252-967e-a04913fe4848",
            "title": "Handmade Granite Computer"
          }
        ]
      }

With parameters:

/api?s=message:lorem.sentence(5);user.name:person.fullName(sex=male) 

      {
        "message": "Lorem ipsum dolor sit amet consectetur.",
        "user": {
          "name": "Michael Johnson"
        }
      }

Complex schema:

/api?s=message:lorem.sentence(5);meta.time:date.anytime;meta.request_id:string.uuid;data[2].id:string.uuid;data[2].name:person.fullName(sex=male);data[2].email:internet.email(firstName=Jeane,provider=vercel.com);data[2].address.street:lorem.word(length.min=5,length.max=7,strategy=fail);data[2].address.city:location.city;data[2].address.country:location.country
